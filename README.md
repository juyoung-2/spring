## Spring Study Repository

이 저장소는 **Spring 학습을 위한 예제와 노트들을 정리한 프로젝트**입니다.  
각 챕터별로 Spring Boot 기반 프로젝트를 통해  
DI, MVC, REST API, AOP, 파일 처리 등  
Spring 핵심 개념을 **단계적으로 학습**할 수 있도록 구성되어 있습니다.

---

## Project Structure

- **chapter01_begin** :  
  Spring 개요, 개발 환경 설정, Spring Boot 프로젝트 생성, 기본 Controller 작성

- **chapter02_DI** :  
  IoC / DI 개념 이해, Bean 등록, `@Component` 계열 어노테이션, 의존성 주입 방식 학습

- **chapter03_MVC** :  
  Spring MVC 구조 이해, Controller / Service / Repository 분리,  
  요청–응답 흐름, View 처리

- **chapter04_REST** :  
  REST API 개념, `@RestController`, JSON 데이터 처리,  
  HTTP Method(GET, POST, PUT, DELETE) 활용

- **chapter05_AOP** :  
  AOP 개념 이해, 공통 관심사 분리,  
  로그 처리 및 메서드 실행 전·후 공통 로직 적용

- **chapter06_File** :  
  파일 업로드 / 다운로드 처리, `MultipartFile`,  
  서버 파일 저장 및 경로 관리

---

## Learning Goals

- Spring Boot 기반 프로젝트 구조 이해
- IoC / DI를 활용한 객체 관리 방식 학습
- MVC 패턴을 적용한 웹 애플리케이션 구조 이해
- RESTful API 설계 및 JSON 통신 처리
- AOP를 활용한 공통 관심사 분리 및 유지보수성 향상
- 파일 업로드 및 다운로드 기능 구현
